CSS Transitions

A value will already be set a list of CSS properties. It will be then given a transition that is a different set of CSS properties, This is the animation effect. Essentially this is the start value that goes to the end value.

Hover over me
.button { color: #4a89ca; border: 2px solid; transition-duration: .4s, .8s, .4s; transition-property: background, border-radius, color; } .button:hover { color:#fff; background: #4a89ca; border-radius: 1em; }

The top color and border defines the start value of the button. The transition duration is ordered inline with the below transition properties. The properties will change to the hover state when hovered over. Then back to the start properties when no longer hovered over.

These are the following transition properties:

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
